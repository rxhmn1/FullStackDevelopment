# 🛒 NovaMart - Premium NovaMart E-Commerce Platform

A modern and responsive **NovaMart E-Commerce Platform** developed as part of the **THIRANEX Web Development Internship - Task 3**.

This project demonstrates a premium online shopping experience with product browsing, filtering, search, cart functionality, user authentication pages, and order management.

## 📌 Features

- 🏠 Premium Home Page with Featured Products
- 🛍️ Dynamic Products Page
- 🔍 Product Search Functionality
- 🎯 Category Filtering
- ↕️ Product Sorting
- 🛒 Shopping Cart System
- 💳 Checkout Page
- 🔐 User Login and Registration Pages
- 📦 Order History Page
- 🖼️ Product Images Displayed Across Home, Products, and Cart Pages
- 📱 Fully Responsive Design
- ✨ Modern Glassmorphism UI Design
