# 📋 TaskPulse - Task Management Application

TaskPulse is a responsive task management web application developed as part of the **Thiranex Web Development Internship**. It helps users organize tasks, track progress, and improve productivity.

## 🚀 Live Demo

🔗 [https://jashwanth018.github.io/thiranex-web-development-internship/Task-2-Task-Management-Application/](https://jashwanth018.github.io/thiranex-web-development-internship/Task-2-Task-Management-Application/)

## 📌 Project Overview

The application allows users to:
- Register and log in securely
- Create, edit, and delete tasks
- Set due dates
- Search tasks
- Mark tasks as completed
- View productivity statistics
- Track overdue tasks with visual indicators
- Manage profile information

All data is stored locally using the browser's Local Storage.

## ✨ Features

- User authentication (Register/Login)
- Dashboard with task statistics
- Add/Edit/Delete task functionality
- Due date tracking with overdue alerts
- Search functionality
- Separate My Tasks and Profile pages
- Responsive glassmorphism design
- Local Storage based persistence

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Responsive Web Design
